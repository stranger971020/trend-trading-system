### 3.2 仓位建议

**市场脆弱度优先**：`danger` 或 `warning` 时跳过追涨区计数，直接按脆弱度定仓位。

| 风险等级 | 仓位上限 |
|---------|---------|
| danger | **≤15%** |
| warning | **≤30%**（轻仓防御，高弹性组合如科技中小盘建议30%） |
| caution | **≤40%**（沿用追涨区计数） |
| normal | 追涨区 ≥8 → 40-50% |
|     | 追涨区 ≥3 → 25-35% |
|     | 追涨区 <3 → 10-20% |

**回补规则**（从减仓状态恢复到正常仓位）：
- 等待首次出现 **行业≥70%上涨**（即≤30%行业下跌）
- 出现后**一步回满**正常仓位，不逐步加仓
- 历史回测显示：立即回满比逐步加仓年化高 **+4%**，交易次数减少 **46%**

> ⚠️ **人工交叉验证（重要）**：当报告出现 `warning` 或 `danger` 信号时，打开券商 app 检查**北向资金流向**。如果北向资金**连续 3 天以上净流出超过 50 亿元**，说明外资也在同步撤退——这种情况下 `warning` 信号的可信度显著提升，**建议直接执行最低仓位（≤15%）**，不等待 `danger` 信号确认。

## 四、板块扫描

### 4.1 四大分区及判定标准

每个 L2 二级行业（约 123 个活跃行业）根据技术指标归入四个分区：

| 分区 | 图标 | 判定条件 | 含义 |
|------|------|----------|------|
| **追涨区** | 🚀 | RSI > 50 + MACD > 0 + 成交量配合 | 趋势向上，可操作性最高 |
| **观察区** | 👀 | MACD 刚转多或 RSI 回升中 | 处于转折初期，等待确认 |
| **高位区** | 🔻 | MACD 转空、RSI 回调或见顶形态 | 趋势走弱或见顶风险 |
| **弱势区** | ⛔ | 不满足上述任一条件 | 无趋势或下降趋势，回避 |

**金融板块过滤**：追涨区排除含"银行/证券/保险/信托/金融/信贷"关键词的板块。

### 4.2 信号标签

每只板块股票附带一组标签：

| 标签 | 含义 |
|------|------|
| `RSI77↑` / `RSI45↓` | RSI 数值 + 方向（↑上升/↓下降/→走平/↗微升） |
| `MACD多` | MACD > 0，多头状态 |
| `MACD空` | MACD < 0，空头状态 |
| `MACD转多` | MACD 刚上穿 0 轴，转折信号 |
| `MACD柱缩` | MACD 柱状图缩短，动能减弱 |
| `量增✅` | 成交量大于 20 日均量 |
| `量缩⚠️` | 成交量小于 20 日均量 |
| `🔴见顶` | 价格新高但 MACD DIF 未创新高（顶背离） |

### 4.3 入场/止损/目标价

追涨区板块附带交易参数：

```
入场: 最近收盘价附近区间
止损: max(MA20最低, 30日最低) × (1 - 缓冲比例)
目标: 入场 + (入场 - 止损) × 盈亏比
```

### 9.2 风险等级与仓位对应

| 报告信号 | 条件 | 总仓位上限 | 操作 |
|---------|------|----------|------|
| ✅ 正常 | 无信号，追涨区正常 | 100% | 维持默认配置 |

| ⛔ 市场脆弱(danger) | 广度崩塌+趋势破位 | **≤15%** | 保留沪深300底仓，其余全清 |

### 9.3 warning 减仓步骤

触发 warning 后，**次日开盘执行**，顺序不可颠倒：

| 步骤 | 操作 | 仓位变化 | 理由 |
|------|------|---------|------|
| 1 | **清掉科创50 ETF（-20%）** | 100% → 80% | 弹性最大，跌最快 |
| 2 | **科技中小盘减半（-20%）** | 80% → 60% | 保留一半防假信号 |
| 3 | **恒生科技减半（-7%）** | 60% → 53% | 港股不完全跟随A股 |
| 4 | 沪深300 不动 | 53% → 53% | 大票抗跌，留底仓 |

最终仓位 ≈ **53%**。若仍高于建议的30%，可再减部分沪深300至目标。

### 9.4 danger 加严

warning 升级为 danger 时：
- 科创50、中小盘、恒生科技 **全部清掉**
- 沪深300 保留 **10~15%** 作为底仓

### 9.5 加仓回补规则（行业≥70%上涨日）

**首次出现行业≥70%上涨当日，一步回满，不分批：**

| 步骤 | 操作 | 仓位变化 |
|------|------|---------|
| 1 | 科创50 ETF → 一步回满 20% | 30% → 50% |
| 2 | 科技中小盘 → 一步回满 40% | 50% → 90% |
| 3 | 恒生科技 → 一步回满 15% | 90% → 105% |
| 4 | 沪深300 调整至 25% | 105% → 100% |

> ⚠️ 回测数据支持：一步回满比分批加仓年化高 +4%，交易次数减少 46%。

### 9.6 提前减仓信号（独立于报告）

**即使报告没有 warning 信号，以下情况也应手动减仓：**

> **北向资金连续 3 天净流出超过 50 亿元 → 提前将仓位降至 50%**

北向资金是外资"聪明钱"的实时风向标。它连续大额流出往往比 A 股广度指标提前 1~3 天。这是一个独立的提前预警，不与报告的 warning 信号绑定。

### 9.7 完整决策流程

```
每日看报告 →
  ├─ warning? → 次日按步骤减仓到 ≤30%
  ├─ danger? → 清仓，留沪深300 10%
  ├─ 行业≥70%上涨？ → 一步回满
  └─ 无信号 + 北向连续3天流出>50亿？ → 提前减到 50%
```
